Context

Clean Path was created as a web development project centered on Charity: Water. Players identify polluted spaces, use flags, track lives, and build a win streak.

Challenges and Growth

The main challenge was making the game behave more like Minesweeper. I added numbered tiles, right-click flagging, difficulty settings, lives, and scoring. This strengthened my ability to divide a complex feature into smaller, testable steps.
