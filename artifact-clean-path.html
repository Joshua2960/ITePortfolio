Clean Path: Charity: Water Game

An interactive browser game inspired by Minesweeper that raises awareness about the global clean water crisis while demonstrating JavaScript programming, game logic, and user interface design.

▶ Play Clean Path View Source Code

Project Context

Clean Path was developed as a web development project inspired by the mission of Charity: Water. The objective was to create an engaging educational experience that encourages players to identify polluted water sources while learning about the importance of clean drinking water. The game combines classic Minesweeper mechanics with a meaningful real-world cause.

Career Relevance

This project demonstrates my ability to build an interactive web application using HTML, CSS, and JavaScript while implementing game logic, event handling, and responsive design. It also showcases my ability to solve programming problems, improve user experience, and iterate on a project based on testing and feedback— skills that are valuable in software development and Information Technology careers.

Challenges and Reflection

One of the biggest challenges was transforming a basic prototype into a game that behaved similarly to Minesweeper. Throughout development I implemented numbered tiles, right-click flagging, a three-life system, multiple difficulty settings, and a scoring mechanic based on win streaks. I also learned how to deploy the project using GitHub Pages and continuously improved the game based on testing and feedback. This project strengthened my JavaScript programming skills while teaching me the importance of iterative development, debugging, and user-centered design.
